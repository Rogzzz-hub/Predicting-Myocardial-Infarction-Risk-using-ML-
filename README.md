# Predicting Myocardial Infarction Risk Using Machine Learning

## Project Overview
This project predicts myocardial infarction risk using machine learning on a dataset of **70,000 records**.

## Supervised Models
- Logistic Regression
- Random Forest
- Gradient Boosting
- SVM
- KNN

## Unsupervised Models
- K-Means
- DBSCAN
- Agglomerative Clustering

## Tools
- Python
- Scikit-learn
- Google Colab
